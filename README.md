# Skirt
Skirt

### Main features
* Bounding box
* Raytrace
* Maps
* Images
* Other entity misc feautures (direction, move, animation)

### Requires
* [Paper](https://papermc.io) highly recommend to use paper instead of spigot 
* [Skript](https://github.com/SkriptLang/Skript)
